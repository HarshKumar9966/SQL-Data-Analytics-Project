Exploratory Data Analysis (EDA) on retail sales data using pure SQL

Performed comprehensive data profiling and insight generation on a multi-table dataset (~60,000 sales transactions, 500+ customers, 200+ products from 2010–2014) entirely in SQL Server.

Key activities:
• Manual database creation and data loading from CSV
• Dimension vs. measure analysis
• Aggregation, grouping, and ranking with window functions
• Breakdown of revenue by country, category, customer, and product
• Identification of top performers and key business metrics
